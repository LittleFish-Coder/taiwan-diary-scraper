# 日期
1933-09-09

# 日記內容

阿江、[[阿願]]、[[阿華]]下午歸去了，大概從前的惡感情可得解消了。[[毛黃趾|岳母]]時常談其家的事情給我們聽，可見她愛撫我們[[不止|不住]]。 欠藥錢的患者未免太久，今日已見三分之二，為之實收入不上十圓。
