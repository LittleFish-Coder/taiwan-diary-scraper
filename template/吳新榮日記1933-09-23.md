# 日期
1933-09-23
# 日記內容
今日祭日，明日禮拜，有二日的休日，所以，雲仔、坤仔，喜喜群來與國山戲遊，是一場好光景了。 自昨夜以來七股路的[[往診]]已達有五回了，可見這裡的消息，依然七股庄是我醫療上的勢力範圍內了。 昨夜頂街[[舉]]挑戰旗下街來了，今晚下街也[[舉]]應戰旗上街去了。這是佳里有名的[[舉]][[暗藝]]的[[起好]]，聽說自四、五年前以來都不見[[舉]]行，而今年已由官憲許可四十日而已。因這[[舉]][[暗藝]]，佳里的商況定然呈出盛勢啦。
